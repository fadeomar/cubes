## Getting Started

1. Open `index.html` in a modern web browser
2. Start adding tasks!

## Browser Support

- Chrome/Edge (recommended - full feature support)
- Firefox
- Safari
- Opera

**Note:** Voice input requires Chrome/Edge. Other features work in all modern browsers.

## Data Storage

Tasks and lists are stored in browser `localStorage`. Your data persists between sessions.

## Usage

### Creating Tasks
1. Click the "Add Task" button
2. Fill in the task details (title is required)
3. Optionally set due date, priority, list, tags, and recurrence
4. Click "Save Task"

### Managing Lists
1. Click the "+" button next to "Lists" in the sidebar
2. Enter a list name and choose a color
3. Click "Create List"

### Filtering & Sorting
- Use the search box to find tasks by keyword
- Select priority filter to show only specific priorities
- Choose sort option to organize tasks differently

### Viewing Statistics
- Click the statistics icon (ðŸ“Š) in the header
- View completion rates, task counts, and overdue tasks
