Want more voices? Install system voices

The voice list comes from your browser/operating system. Install more English voices and refresh the page.

Windows 10/11

  • Settings ‚Üí Time & language ‚Üí Language & region
  • Add a language (e.g., English US/UK) ‚Üí Language options
  • Download Speech (and Text-to-speech if shown)

macOS

  • System Settings ‚Üí Accessibility ‚Üí Spoken Content
  • System Voice ‚Üí Manage Voices ‚Üí download more English voices

Android

  • Settings ‚Üí System ‚Üí Languages & input ‚Üí Text-to-speech output
  • Install / download additional voices

After installing voices, click Reload voices (or refresh the page).
